👨‍💼 Employees Management System

This project is a backend management system built using Django and Django REST Framework that performs CRUD operations on Movies and Employees data.

🚀 Features:

* CRUD operations for Movies
* CRUD operations for Employees
* REST APIs for both modules
* JSON response handling
* Scalable project structure

🛠️ Tech Stack:

* Python
* Django
* Django REST Framework
* SQLite

📌 Purpose:
To understand how to manage multiple models and build RESTful APIs in a structured way.

📂 Use Case:
This system can be used as a backend for applications that need to manage media content and employee data.
